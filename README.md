# AI Image Prompt Generator

Turn a simple description into a detailed prompt for AI image generators like Midjourney, Stable Diffusion, or DALL-E. No API needed — it works by combining your description with style, mood, lighting, and composition modifiers.

## How to Use

1. Open `index.html`
2. Type a simple description — "a cat sitting on a chair"
3. Pick a style (photorealistic, oil painting, watercolour, pixel art, anime, 3D render)
4. Pick a mood (warm, cool, dramatic, peaceful, dark, vibrant)
5. Click "Generate Prompts"
6. Get multiple detailed prompt variations
7. Copy the one you like best

## What It Adds

The generator enriches your simple description with:
- **Lighting** — golden hour, studio lighting, neon glow, candlelight, etc.
- **Composition** — rule of thirds, close-up, wide angle, bird's eye view
- **Camera details** — for photorealistic styles: lens, aperture, film stock
- **Artistic references** — for painting styles: techniques, texture descriptions
- **Quality modifiers** — highly detailed, 8k, masterpiece, etc.
